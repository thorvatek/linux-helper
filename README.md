## ğŸš€ Installation  

1. **Clone the repository**:  
   ```bash  
   git clone https://github.com/thorvatek/linux-helper.git  
   cd linux-helper  
   ```  

2. **Make the script executable**:  
   ```bash  
   chmod +x install-packages.sh
   ```  

   ```bash

   ```
