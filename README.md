Fedora Linux Helper Tool that automates the setup and configuration of an fedora Linux system. It installs and configures packages, and various utilities to create a fully functional gaming and production environment.

## Prerequisites
- A fresh Fedora Linux installation
- Internet connection
- sudo privileges

## Installation
1. Clone this repository:
   ```
   git clone https://github.com/thorvatek/linux-helpper.git
3. Run the setup script:
   ```
   ./install-packages.sh
4. Follow the prompts to select the packages you want to install.
5. The script will handle the rest of the setup process.
6. After the setup is complete, you can reboot your system to see the changes.
