Fedora Linux automation tool for system setup and configuration. Streamlines the installation and configuration of packages and utilities to establish a comprehensive gaming and production environment.

## Prerequisites
- A fresh Fedora Linux installation
- Internet connection
- sudo privileges

## Installation
1. Clone this repository:
   ```
   git clone https://github.com/thorvatek/linux-helpper.git
2. After downloading the script, execute the following command:
   ```
   chmod +x install-packages.sh
3. Run the setup script:
   ```
   sudo ./install-packages.sh
4. Once the script finishes, reboot your system to see the changes.
   ```
   sudo reboot
